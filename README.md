# WYRM

**Площадка коллективного повествования** — истории, которые сообщество пишет вместе:
глава за главой, развилка за развилкой. Каждая история живёт как дерево ветвей,
а лучшие ветви становятся каноном.

> Проект изначально был собран в Claude Design как один HTML-бандл. Здесь он
> распакован в полноценный **Vite + React** репозиторий, с добавленными лентой
> и сообществами и доведёнными до рабочего состояния механиками.

## Запуск

```bash
npm install      # установка зависимостей
npm run dev      # дев-сервер на http://localhost:5173
npm run build    # прод-сборка в dist/
npm run preview  # предпросмотр прод-сборки
```

## Что внутри

- **Главная** — лендинг с витриной флагманской истории.
- **Лента** — поток активности сообщества: новые ветви, голосования, споры за канон.
- **Каталог** — все истории с фильтром по настроению (тегам) и сортировкой.
- **Сообщества** — тематические кружки авторов: создание, вступление, своя витрина историй.
- **Древо** — интерактивная карта ветвей истории (зум, перетаскивание, выбор канона).
- **Писать** — редактор новой главы/ветви с указанием судеб персонажей.
- **Студия** — 5 механик: Слияние, Кодекс мира, Канон-ставки, Комната авторов, Сборка читателя.
- **Расширения** — магазин и конструктор плагинов сообщества.
- **Кастомизация** — два мира оформления (Ночь / Манускрипт), акценты, шрифты, масштаб.

## Архитектура

Без серверной части: всё состояние живёт в браузере (`localStorage`, ключи `wyrm.*`).

```
index.html              # точка входа Vite
src/
  main.jsx              # бутстрап: стили + приложение
  styles.css            # дизайн-система (две темы, токены, типографика)
  app.jsx               # приложение (компоненты, маршрутизация, данные)
public/fonts/           # шрифты Onest / Lora / JetBrains Mono (woff2)
```

Маршрутизация — простое состояние `route` в `App` с функцией `go(route, payload)`;
кросс-экранные данные передаются через объект `ctx`.
